# 📧 Simple Email System (Python OOP)

A simple console-based email simulation built using Python OOP concepts. It allows users to send, receive, read, and delete emails.

---

## 🚀 Features

- Create users with inboxes
- Send emails between users
- View inbox messages
- Read full emails
- Delete emails
- Track read/unread status
- Automatic timestamps

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Class relationships (User → Inbox → Email)
- Encapsulation
- Python `datetime`
- List-based data storage

---

## ▶️ How to Run

```bash
python email_system.py
